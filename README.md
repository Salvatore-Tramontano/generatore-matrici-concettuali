# Generatore di matrici concettuali

Il generatore di matrici concettuali è uno strumento per creare, visualizzare ed esportare matrici di posizionamento.

L'intera applicazione è contenuta in un singolo file HTML. Non richiede installazione, server o connessione internet per funzionare. È ideale per visualizzare rapidamente concetti strategici (es. matrici di posizionamento, analisi competitor, SWOT visive).

## Caratteristiche Principali
Zero Installazione: Basta scaricare il file e aprirlo nel browser.
Interattività Totale: Posiziona gli elementi con il drag & drop.
Immagini Personalizzate: Carica loghi o foto per ogni punto della matrice (con tool di ritaglio incluso).
Esportazione HD: Scarica il grafico finale come immagine .png ad alta risoluzione.
Salvataggio Progetti: Salva il tuo lavoro in formato .json per riprenderlo in seguito.
100% Privacy: I dati rimangono nel tuo browser. Nulla viene inviato a server esterni.
Design Adattivo: Funziona su desktop e ha un'interfaccia ottimizzata per tablet/mobile.
